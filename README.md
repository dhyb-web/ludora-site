# Site Ludora

Ce dossier contient le portail statique de Ludora :

- `index.html` : site officiel avec lien APK, catalogue, Ludora Football et roadmap.
- `blog.html` : devlog, roadmap, backend et articles football.
- `support.html` : installation APK, Supabase et FAQ.
- `legal.html` : confidentialite, conditions et base support.

Le site peut etre ouvert directement dans un navigateur depuis `index.html`.

Pour publier gratuitement :

1. Envoyer le dossier `site` sur GitHub Pages, Netlify ou Vercel.
2. Remplacer les liens de support par une URL officielle de telechargement APK quand elle est prete.
3. Remplacer les textes de support par une vraie adresse email avant publication publique.
