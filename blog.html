Devlog31 mai 2026

Les modules avances deviennent vraiment actifs.

La plateforme ne se contente plus d'afficher un message quand un module est lance. Chaque action sociale, gaming, monetisation, securite, IA, technique, live ou UI cree maintenant une activite visible.

Ce qui change

Ludora sauvegarde les actions localement, ajoute XP et LudoCoins quand le module le permet, garde un historique par section, puis tente une synchronisation Supabase via ludora_module_states.

Impact joueur

Quand tu lances une invitation, une mission, un coffre, un scan anti-triche ou un test IA, tu vois l'activite du module directement dans l'interface.

FootballMVP

Ludora Football : une seule interface, plusieurs modes dedans.

Le jeu football sert de vitrine sportive a Ludora. L'objectif n'est pas encore de remplacer Unity, mais de construire un MVP mobile clair, jouable et extensible.

Actif maintenant

Le match contient une vue 2D tactique, une vue 3D stylisee, le changement de joueur, les tirs, passes, dribbles, pressing, fautes simples, corners, coups francs, meteo, mini-carte et tournoi.

Prochain objectif

Le prochain palier consiste a connecter les invitations amis au mode match, creer des salons football, puis synchroniser les statistiques de match.

BackendSupabase

Pourquoi le SQL doit etre relance apres les nouveaux modules.

Quand Ludora gagne une nouvelle table ou une nouvelle policy RLS, Supabase ne peut pas la deviner. Le script SQL complet met a jour la base sans effacer les donnees existantes.

Tables importantes

ludora_players, ludora_scores, ludora_game_sessions, ludora_friendships, ludora_online_rooms, ludora_platform_events et ludora_module_states.

RoadmapProduit

Le plan pour rendre Ludora plus vivant.

La priorite court terme est de rendre les actions visibles, testables et synchronisables. Ensuite, les modes amis et tournois doivent devenir le centre de l'experience.

Priorites

Compte email fiable, salons amis, jeux plus profonds, missions quotidiennes, saisons, store d'objets, classement mondial temps reel et dashboard web admin.
